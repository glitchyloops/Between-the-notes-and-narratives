# Between-the-notes-and-narratives
Between the notes and narratives: A dynamic mood explorer is a responsive web-based  platform that provides personalized music and literary recommendations based on a user's  emotional state.

## What it does

* Lets users select a mood (happy, melancholic, energetic, romantic, calm, mysterious, fierce, nostalgic)
* Recommends **songs** from a curated dataset
* Fetches **book suggestions dynamically** using the Google Books API
* Allows users to **submit their own recommendations**

## How it works

* The interface responds to your chosen mood with visual themes and curated content
* Songs are selected from predefined mood-based collections
* Books are fetched in real-time based on mood-related search queries
* Smooth UI interactions and animations create an immersive experience

## Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **API Integration:** Google Books API
* **Design:** Custom CSS with dynamic mood theming and animations

## Project Structure

* `index.html` – Structure and layout
* `styles.css` – Visual design and mood-based themes
* `script.js` – Logic, interactivity, API handling
